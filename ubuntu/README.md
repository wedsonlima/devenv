## Ubuntu dev environment

*From github page:*
```
$ \curl -sSL https://raw.githubusercontent.com/wedsonlima/devenv/main/ubuntu/startup.sh | bash
```

*Or:*

```
$ wget -O startup.sh https://raw.githubusercontent.com/wedsonlima/devenv/main/ubuntu/startup.sh
$ chmod +x startup.sh
$ sh startup.sh
```
